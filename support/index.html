Support

If you run into a problem while using Runway, want to report a bug, or have a question about what the product is doing, you can contact us directly. This page is intended to be clear enough for users and explicit enough for App Store review.

Helpful details to include

  • Relevant screenshots or screen recordings
  • If the issue involves ledger data, trends, or runway calculations, a brief description of the numbers involved
